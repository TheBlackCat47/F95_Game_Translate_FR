Politique de Confidentialité
Dernière mise à jour : 19 avril 2026 · Applicable à : Game Translate FR (site web & bot Discord)

1. Présentation

Game Translate FR est un projet communautaire bénévole qui recense les traductions françaises de jeux disponibles sur F95Zone. Ce projet comprend un site web statique hébergé sur GitHub Pages, un bot Discord d'information et un service de synchronisation automatique des données.

La présente politique décrit comment nous traitons les données dans le cadre de ce projet. L'utilisation du site et du bot Discord implique l'acceptation de cette politique.

2. Données collectées par le site web

✓ Sans compte Le site web ne nécessite aucune inscription, aucun compte et aucune connexion.

Le site web ne collecte aucune donnée personnelle identifiable. Nous n'utilisons pas de cookies de traçage ni de formulaires.

Les préférences locales (filtres, mode clair/sombre) sont stockées uniquement dans le localStorage de votre navigateur et ne sont jamais transmises à nos serveurs.

3. Analytique (mesure d'audience)

Le site utilise deux outils d'analyse d'audience respectueux de la vie privée :

  • Umami (umami.is) — collecte des statistiques anonymes (pages vues, pays, navigateur). Aucun cookie déposé, aucune donnée personnelle stockée, conforme RGPD. Hébergé sur cloud.umami.is.
  • GoatCounter (goatcounter.com) — compteur de visites anonyme, sans cookies, sans suivi inter-sites.

Aucun de ces outils ne permet d'identifier personnellement un visiteur.

4. Hébergement et infrastructure

  • GitHub Pages — Le site est hébergé via GitHub Pages (Microsoft). Les accès génèrent des logs serveurs standard (adresse IP, user-agent) conservés par GitHub conformément à leur politique de confidentialité.
  • Serveur de synchronisation — Un serveur privé interroge l'API Google Sheets toutes les 15 minutes pour mettre à jour le catalogue. Ce serveur ne traite aucune donnée relative aux visiteurs.

5. Sources de données du catalogue

Les données affichées proviennent de :

  • Google Sheets — Tableur communautaire maintenu par des bénévoles. Ces données sont publiques et accessibles à tous.
  • F95Zone — Images de jaquette et métadonnées des jeux récupérées depuis f95zone.to via son flux RSS public.

Ces données concernent des œuvres (jeux, traductions) et non des personnes. Les noms de traducteurs présents dans le catalogue sont des pseudonymes choisis volontairement par leurs auteurs pour être publics.

6. Bot Discord

Le bot Discord associé à ce projet (bot de notification communautaire) opère dans les serveurs Discord où il est invité. Voici ce qu'il collecte et traite :

Ce que le bot collecte :

  • Les identifiants de serveurs (guild ID) et de salons (channel ID) pour l'envoi des notifications.
  • Aucun message privé, aucun historique de conversation, aucun contenu utilisateur.

Ce que le bot ne collecte pas :

  • Noms d'utilisateurs, adresses e-mail, données de profil Discord.
  • Messages, réactions ou activités des membres du serveur.
  • Données biométriques ou de localisation.

Utilisation des données Discord :

  • Les identifiants de serveurs/salons sont utilisés exclusivement pour envoyer les notifications de mises à jour du catalogue.
  • Ces identifiants ne sont jamais partagés avec des tiers.
  • La suppression du bot du serveur entraîne la suppression de toute configuration associée.

Durée de conservation : Les données sont conservées tant que le bot est présent dans le serveur. Elles sont supprimées sur simple demande ou lors du retrait du bot.

7. Vos droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Accès — droit de savoir quelles données vous concernant sont traitées.
  • Rectification — droit de faire corriger des données inexactes.
  • Suppression — droit à l'effacement des données vous concernant.
  • Opposition — droit de vous opposer au traitement de vos données.

Pour exercer ces droits, contactez-nous via Discord (voir section 8).
